1 : Ground. 

 We first create a variable for the rigid body,i.e. , the ground. This shape is a straight line with 2 end-points -90.0f and 90.0f.
 After that we define a BodyDef object which is passed to the world object to create a body. 
 Finally the ground is shown using CreateFixture which has zero density and is now cisible on the GUI

2 : Top horizontal shelf 

 We create a rectangle shape with width 12.0f and 0.5f and the center of this object is at (-31,30) from the attributes of bd.
  Finally, we show the object on the GUI using CreateFixture

--------------------------

3 : Dominoes 

We first set a shape which is a rectangle with width 0.2f and height 2.0f.Then a fixture is defined fd 
    with density 20.0f and co-efficient of friction to be 0.1
  We then set up 10 dynamic bodies(dominoes) with y-co-ordinate as 31.25f and the x-co-ordinates vary 
  from -35.5f to -26.5f. The objects are finally shown using CreateFixture with the b2Body object 
  which is the horizontal platform.

7: The pulley system
This block creates a dynamic body which is set to position (-10,15) and it is allowed to rotate while being fixed to a point

The open box is created using 3 rectangles joined to form an open box

The open box is set to be shown with certain attributes like density to be 10,friction co-efficient as 0.5

Co-effecient of restitution to be 0

The shape is declared which is then later defined to be a rectangle of width 4,height 0.4, center of box to have co-ordinates to be(0,-1.9) and density 0

Another rectangle is defined to have same density,friction and restitution as the previous rectangle but the width is 0.4,height is 4, rectangle's center is located at (2,0) with density 0

Another rectangle is defined to have same density,friction and restitution as the previous 2 rectangles but the width is 0.4,height is 4, rectangle's center is located at (-2,0) with density 0

The 3 boxes are joined together using CreateFixture and shown.

The bar is a rectangle with it's center's co-ordinates to be (10,15), density 34 and then the bar is displayed on screen using CreateFixture

8:The revolving horizontal platform

A rectangle is created with width 4.4f and height 0.4f

A dynamic body is created with it's center's co-ordinates to be (14,14)

The dynamic body is given a density of 1 and it's shape is set to that of the rectangle created above.

The dynamic body is then shown on the GUI using CreateFixture

A joint is created so that the horizontal platform can rotate about a hinge

The joint joins the horizontal platform and an imaginary body and the point of rotation is set to be (0,0)

The 2 joined bodies are not allowed to collide with each other
