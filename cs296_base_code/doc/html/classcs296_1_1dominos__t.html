1 : Ground. 

 We first create a variable for the rigid body,i.e. , the ground. This shape is a combination of three straightlines arranged in such a way that it would hold the container of balls (when it falls down).three fixtures are created for three straight lines and they are passed to body
        we define a BodyDef object which is passed to the world object to create a body. 
        Finally the ground is shown using CreateFixture which has zero density and is now visible on the GUI

-------------------------

2 : Top horizontal shelf 

 We create a rectangle shape with width 6.0f and 0.5f and the center of this object is at (-39,45) from the attributes of bd.
  Finally, we show the object on the GUI using CreateFixture

-------------------------

3 : small and a large ball combination which sets off the simulation

<prev> we first create a circle with 0.8 radius and hten create a larger circle with 1.8 radius. we link these two bodies with a pulley joint so that as the bigger ball goes down the smaller one hits the vertical hinge and sets off the simulation. </prev>

-------------------------

3 : the vertical hinged box

we first create b2Body* object "hinge" (Box shape) which won't be displayed as we don't assign any fixture to it.

the b2Body object "rod" is also created in a similar fashion.this will be displayed as we assign the fixture "rodFixtureDef" to this.

then we create a b2RevoluteJointDef object "jointDef" and link the rod and hinge and we constrain the upper limit and lower limit of rotation angles appropriately.

-------------------------

4 : the newton's cradle

first we create a ceiling from which balls are hanged.

b2Body* object "bar" is created as a box with dimensions 7x0.04

consequently fixture is defined for "bar"

In the "for" loop we create four pendulum which will be placed side by side to produce newton's cradle The pendulum is essentially a single ball which is anchored to ceiling to produce pendulum effect.

A b2Body* object bob is created in the shape of a circle.

As u can see the final bob is given more density as it involves in a collision with a ball subsequently.

b2Body* "hinge" is created with no fixture definition so that it on't be displayed.

a b2RevolutejointDef "revJointDef" is created which is used to link the "hinge" and the "ball"

-------------------------

5 : The big blue boulder
this is the ball which will be set in to motion by the newton's cradle.

a b2FixtureDef object "boulderFixtureDef" is created the shape assigned as a circle with radius 1.2f,density = 1.5f, friction = 80.5f; restitution 0.0f

now we have to create a stationary bar which holds the above "blueBoulder"

a b2Body* object boulderGrdBody is created. it is set as a box with dimensions 4x0.2

-------------------------

5 : the open box

The open box is created using 3 rectangles joined to form an open box

we declare three fixture definitions and assign all the three to a b2Body object to produce a openbox

Co-effecient of restitution to be 0

now we build a vertical bar which is linked to the openbox by a pulley

a b2Body object "box2" is created for creating the bar and we assign the fixture subsequently.

The pulley joint

Anchor point for ground 1 in world axis

Anchor point for ground 2 in world axis

we define the ratio and create the pulley joint

a new open box is created exactly as shown above.

The open box is created using 3 rectangles joined to form an open box

The 3 boxes are joined together using CreateFixture and shown.

-------------------------

6 : the balls on the static slant platform

Then we create 4 balls and position them on the slant static platform

4 b2Body* objects are created for this purpose (dynamicBody,dynamicBody1,dynamicBody2,dynamicBody3)

-------------------------

7 :static slant platform
we just create a box at an angle which is stationary

-------------------------

8 :the see saw
first we create the traingular wedge which supports the plank above it. we define the verticees for the shape to create a trinagle.

task1 -- The triangle wedge

sbody -- b2Body variable for wedge

shape declared as polygon.

vertices -- b2vec2 arrayv which contains the vertices for wedge.

vertices[0] set to (-1,0).

vertices[1] set to (1,0).

vertices[2] set to (0,1.5).

wedgefd -- b2FixtureDef variable for fixture definition.

shape defined as polygon.

density set to 10.0f

friction set to 0.0f

restitution set to 0.0f

wedgebd -- b2BodyDef varaible.

position set to -26.0f,10.5f

triangle wedge is created.

task2 -- The plank on top of the wedge

shape set to box with dimensions (12.0f,0.2f).

bd2 -- b2BodyDef variable defined.

position set to (-26.0f , 12f)

the type of the body is set to b2_dynamicBody as it will be moving.

fd2 -- pointer to b2FixtureDef variable.

density set to 3.f

shape set to polygon

plank object is created.

jd -- b2RevolutejointDef variable for joint.

anchor declared.

anchor position set to (-26.0f, 12f)

joint is intialised with the menbers anchor,body , sbody.

8 :the small ball on the see-saw
we create a small ball and position it on the plank we created above

we turn off gravity for this ball using SetGravityScale(x) function.

-------------------------

9 :the 4 balls on the horizontal platform with ditches.

in this part we just create three circles and position them appropriately.

--------------------------

10 : Dominoes 

We first set a shape which is a rectangle and we assign a fixture to it.  We then set up 10 dynamic bodies(dominoes). we then create a larger domino at the end.

--------------------------

10 : revolving platforms
we just create 4 bars which are anchored at the centre for this part.

we create a horizontal bar and another bar which won't be visilble as we don't assign a fixture definition for it. and we anchor both of them to get a body which is free to rotate about it's centre

--------------------------

10 : platform with pits.
we have designed this in the following way. we created two horizontal bars which are on the same level but seperated from each other and we kwpt two pits one in between the two and other at the end. the first pit can accomodate 2 balls and the second pit accomodates 1 ball. these pits are created in the similar way we have created the open box.

the first snippet is to create the first horizontal bar

the second snippet creates the second horizontal part.

the third snippet creates the first open box. the depth of this pit is such that it can accomodate two balls.

the fourth snippet creates the second open box. the depth of this second pit can accomodate one ball.

--------------------------

10 : Conveyer Belt
the way in which we designed the conveyer belt is the following. first we create 4 chains and then link them end to end to forma rectangle (sort of) then we insert three circles spaed vertically inside the chain. these two circles are given a constant angular velocity and they are kinematic so they won't stop. these balls are given sufficient friction so that they drag the chain along in turn creating a conveyer belt. we attach two 'L' shaped fixtures to two links which are diagonally opposite.

--------------------------
